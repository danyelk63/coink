## Coink registro v1.0.0

# Instrucciones

1. Descargar usando git clone *urlDelProyecto*
2. Instalar java 23
3. Instalar java 17
4. Instalar AndroidStudio
5. Ejecutar en la raiz del proyecto ```npm i```
